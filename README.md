# Ramayana

_Shrimad Valmiki Ramayanam_

[![Live Demo](https://img.shields.io/badge/Live%20Demo-Coming%20Soon-C9A84C?style=for-the-badge)](#)

A dark cosmic, Sanskrit-inspired frontend tribute to the Ramayana. The project presents an immersive character archive and a seven-book story experience built with vanilla HTML, CSS, and JavaScript.

## Features

- Cinematic character constellation with searchable and filterable Ramayana figures.
- Photo-backed character cards with graceful CSS gradient fallback.
- Dedicated story page covering all seven Kandas with tabbed navigation.
- Redesigned manuscript-style episode cards with gold accents, Roman markers, hover lift, and read-more reveal.
- Sacred Summary banners using Sanskrit shlokas from the page content.
- Horizontal Cosmic Timeline linking directly to each Kanda.
- Deep links such as `Story.html#bala`, `Story.html#sundara`, and `Story.html#uttara`.
- Responsive, accessible HTML with keyboard-friendly cards, modals, and tab controls.

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Google Fonts: Cinzel Decorative, Cinzel, EB Garamond

## Local Usage

Open `index.html` in a browser to explore the character archive. Open `Story.html` for the Kanda story experience, or navigate directly to a Kanda with a hash link such as:

```text
Story.html#ayodhya
```

## Sacred Sources

This project is a devotional and educational tribute inspired by the Valmiki Ramayana, traditionally attributed to Maharishi Valmiki. Sanskrit verses and Kanda references are credited to the sacred epic tradition of the Ramayana.
